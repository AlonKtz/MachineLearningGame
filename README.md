# 🤖 אַמֵּן את ה-AI · Train the AI

משחק דפדפן קצר שמלמד את ההבדל בין שלושת סוגי למידת המכונה, דרך התנסות פעילה.
A short browser game that teaches the three types of machine learning by letting you train each one yourself.

## 🎮 שלוש התחנות / Three stations

| תחנה | סוג | מה עושים |
|------|-----|----------|
| 🍿 מנוע המלצות | **Supervised** · מונחית | מדרגים כתבות, והמכונה לומדת את הטעם שלך ומנבאת מה עוד תאהב |
| 👥 פילוח לקוחות | **Unsupervised** · לא-מונחית | המכונה מגלה לבד קבוצות לקוחות מתוך הנתונים (k-means) |
| 🤖 רובוט במבוך | **Reinforcement** · חיזוק | הרובוט לומד לצאת ממבוך בניסוי וטעייה (Q-learning) |

הרעיון המרכזי: ההבדל בין הסוגים נקבע לפי שאלה אחת — **מה נתנו למכונה?** תוויות, כלום, או תגמול.

## ▶️ איך מריצים / How to run

פשוט פותחים את `index.html` בכל דפדפן. אין צורך בהתקנה.
Just open `index.html` in any browser. No installation needed.

הכול בקובץ HTML יחיד, כולל מימוש אמיתי של האלגוריתמים (לא הדמיה).
Everything is in a single HTML file, with real algorithm implementations.
